Hashing

It is a searching technique based on concept of hash table.
It does not require unnecessary comparisons and directly find the location of the number in one access.
In this method, the location of derived record present in search table depends only on the given key eg. Table with record of n students, each of which is defined by roll no. key.
Hashing reduces comparisons than binaery search.

Hash function:-
A function that is used to compute the address of a record in the hash table. A hash function transfers X identifier into bucket address in hash table.
the address computed is called as hash address of identifier X.
If more than one record have same hashing address, then there is collision occur. this is called address collision

Properties of hash function:-

It should be easy to compute.
2.It should minimize no of collisions.

Searching

Searching means finding derived element in given list of elements.
There are two types of searching
1.Linear Search.
2.Binary Search

1.Linear Search:-

Basic and simple method of searching on this search, we access each element of an array one by one sequencially and see whether it is derived.
A searc wull be unsuccessful if all the elements are accessed and the derived element is not formed.
It is also called as sequential search.
If the derived element is present in 1st position of the array , then only are are comparison is reduced.
Best case complexity- O(1).
If the derived element is present in the last position,then n comparisons are required
Worst case complexity- O(1)

2.Binary Search

Extremely efficient algorithm.This algorithm searches element in minimum possible comparisons.
The only drawback is that here list of elements must be saved.
There are 3 cases
i) If it is derived element then search is successful.
ii) If it is less than derived item then search only in the first half of the array.
iii) If it is greater than derived item then search only in the 2nd half of the array.
Repeat these steps until an element is found pr array is over .
In this searching,we are reading the array in every step
Worst case complexity- log(n+1)

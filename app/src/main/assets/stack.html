Stack

A stack is a basic data structure that can be logically thought of as a linear structure represented by a real physical stack or pile, a structure where insertion and deletion of items takes place at one end called top of the stack. The basic concept can be illustrated by thinking of your data set as a stack of plates or books where you can only take the top item off the stack in order to remove things from it. This structure is used all throughout programming.

The basic implementation of a stack is also called a LIFO (Last In First Out) to demonstrate the way it accesses data, since as we will see there are various variations of stack implementations.

There are basically three operations that can be performed on stacks. They are
1) inserting an item into a stack (push).
2) deleting an item from the stack (pop).
3) displaying the contents of the stack.

1.Push:

The process of adding new element to the top of the stack is called Push Operation.
for every pushing element,top of stack is incremented by one
If array is full and no element can be inserted, that condition is called stack full Or Stack overflow condition

Algorithm for Push Operation:-

    Push(Stack [Maxsize],item)
    let Stack[Maxsize] be an array for implementing stack
    1.[check if stack overflow]
    If Top=Maxsize-1
        Print "Overflow"
    else
        Top=Top+1
        Stack[Top]<-item
    2.Exit

2.Pop:

pop() function is used to remove an element from the top of the stack(newest element in the stack).
The element is removed to the stack container and the size of the stack is decreased by 1.
If there is no element in stack and the pop is performed then this will result into Stack Underflow Or Stack EmptyCondition

Algorithm for Pop Operation:-

    Pop(Stack [Maxsize],item)
    1.[check for stack underflow/empty]
    2.if(Top==-1)
        print "stack underflow"
    else  [Remove the top element]
        Top=Top-1
        Return the deleted item
        item=item=stack[Top]
    3.Exit
